### STK Code Snippets
this is a collection of various Python, Matlab, C# and vb.Net code snippets. Some of these originally came from the STK Programming Help but were reorganized and appended. 

The code snippet layout is designed to make it easier to find the code snipptes you need as it tries to match the GUI layout. For example the code snippets for setting up the initial state of an HPOP satellite are in 
STK Objects\Satellite\Basic\Orbit\HPOP

note the STK Objects\Common folder which contains code snippets that apply to many object types like access constraints
